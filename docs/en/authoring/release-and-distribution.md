# Release and distribution

Registries should publish immutable app releases with manifest hash, version, channel, owner, compatibility, and rollback target. Tenant enablement should pin a release or channel. Releasing a new version must not overwrite customer overlays.

## Checklist

- Keep declarations machine-readable.
- Keep procedures in Agent Skills.
- Keep facts in Agent Knowledge.
- Keep execution in the host runtime.
- Attach app provenance to projected objects.
