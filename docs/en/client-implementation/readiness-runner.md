# Readiness runner

The readiness runner checks installed Skills, bound Knowledge, granted Tools, Artifact support, Eval availability, runtime target, and policy. It should produce actionable setup tasks, not opaque failure messages.

## Checklist

- Keep declarations machine-readable.
- Keep procedures in Agent Skills.
- Keep facts in Agent Knowledge.
- Keep execution in the host runtime.
- Attach app provenance to projected objects.
