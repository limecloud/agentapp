# JSON Schemas

The reference schemas are published under docs/public/schemas. They cover app manifest, app projection, and readiness result. They are intentionally permissive in v0.1 so hosts can experiment while preserving required fields.

## Checklist

- Keep declarations machine-readable.
- Keep procedures in Agent Skills.
- Keep facts in Agent Knowledge.
- Keep execution in the host runtime.
- Attach app provenance to projected objects.
