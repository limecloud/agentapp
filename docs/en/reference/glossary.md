# Glossary

| Term | Meaning |
| --- | --- |
| Agent App | Installable app-like package for an agent host. |
| Host | Product that installs, resolves, and executes apps, such as Lime Desktop. |
| APP.md | Required entry file with manifest frontmatter and Markdown guide. |
| Entry | Host-visible scene, command, home, workflow, or artifact surface. |
| Projection | Deterministic host catalog generated from an app manifest. |
| Overlay | Tenant, user, or workspace customization applied over app defaults. |
| Readiness | Static setup check before running an app entry. |
| Provenance | App identity, version, manifest hash, and source refs attached to projected objects. |
