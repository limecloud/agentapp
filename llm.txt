## Core boundary

Agent App composes Agent Skills, Agent Knowledge, Agent Tool, Agent Context, Agent UI, Agent Artifact, Agent Evidence, Agent Policy, Agent Runtime, and Agent QC into an installable app package. It does not run agents by itself.
